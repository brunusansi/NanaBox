# Host-Side Automation Scripts

This directory contains PowerShell scripts and helper utilities that run on the host system to automate anti-detection configuration.

## Purpose

Host-side scripts are responsible for:
- Extracting hardware information (SMBIOS, ACPI tables, etc.)
- Generating realistic spoofing configurations
- Creating and managing VHD/VHDX files
- Configuring network adapters and MAC addresses
- Multi-account VM management

See full documentation at the end of this file for script templates and examples.
